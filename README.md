# docker-jim

## learn docker in 40 minutes

**docker build . -t first_image:1.0**

**docker images**

**docker run first_image:1.0**

**docker ps**

**docker stop container-name|container-id**

**docker start container-name|container-id**

**docker ps -a (show all container also stop containers)**

**docker run -d first_image:1.0 (detach mode)**

**docker logs container-id**
